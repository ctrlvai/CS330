CS 330 – Computational Graphics and Visualization

Project Overview

This project demonstrates my ability to design and develop a 3D scene using OpenGL. The scene was created using basic shapes, textures, and lighting to recreate a real-world setup. It highlights my understanding of transformations, rendering, and lighting techniques.

Reflection

How do I approach designing software?

When approaching software design, I focus on breaking the problem into smaller, manageable parts. For this project, I started by analyzing the reference image and identifying which basic shapes could represent each object. This made the process more organized and easier to build step by step, instead of trying to create everything at once.

What new design skills has this project helped me develop?

This project helped me improve my ability to translate real-world objects into simplified digital representations. I also developed a better understanding of spatial awareness, object placement, and how lighting impacts the overall design of a scene.

What design process did you follow?

I followed a step-by-step approach:

-Build basic shapes

-Position objects in the scene

-Apply textures

-Add lighting

-Refine details

This iterative process helped me gradually improve the scene instead of trying to perfect everything at once.

How could this design approach be applied in the future?

This approach can be applied to future projects by breaking down complex problems into smaller steps. It is especially useful in UI design, game development, and any visual-based application where structure and layout matter.

How do I approach developing programs?

I approach development by starting with a working base and then gradually adding features. I test frequently to make sure everything works before moving on. This helps prevent large errors and makes debugging easier.

What new development strategies did you use?

One key strategy I used was separating logic into functions, such as rendering and transformations. This made the code easier to read and maintain. I also focused on making small incremental changes instead of large ones.

How did iteration factor into your development?

Iteration played a major role in this project. I frequently adjusted object sizes, positions, and lighting values to improve the final result. Many parts of the scene required trial and error before they looked correct.

How has your approach evolved throughout the milestones?

At the beginning, my focus was mainly on getting objects to appear. As I progressed, I became more focused on realism, lighting, and organization. My code also became more structured and easier to follow.

How can computer science help me reach my goals?

Computer science provides me with problem-solving skills and technical knowledge that can be applied in many different careers. It allows me to build solutions, work with data, and create interactive systems.

How do computational graphics help in your education?
Computational graphics helped me understand how visual systems are built and how math is applied in real-world scenarios. It also strengthened my understanding of transformations and rendering.

How do computational graphics help in your career?
These skills can be applied in areas like product design, data visualization, and software development. Understanding graphics and visualization makes it easier to communicate ideas visually and build more interactive applications.
